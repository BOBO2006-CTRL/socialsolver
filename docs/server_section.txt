СЕРВЕРНАЯ ЧАСТЬ ПРИЛОЖЕНИЯ

Серверная часть платформы SocialSolver реализована с использованием языка программирования Python и фреймворка Django.
Для обработки HTTP-запросов и взаимодействия с клиентской частью используется архитектура REST API.

В качестве серверного приложения применяется Gunicorn, который обеспечивает стабильную работу Django-приложения.
Для хранения данных используется система управления базами данных PostgreSQL.

Развёртывание серверной части осуществляется с использованием технологии контейнеризации Docker,
что позволяет обеспечить переносимость, воспроизводимость среды и упрощённый запуск приложения.
